### Projetos

**[BringIA Platform](https://github.com/devmarrima)**
Plataforma de recrutamento distribuída com 6+ microsserviços Quarkus, pipeline Kafka, workers de coleta resiliente, enriquecimento por IA (Anthropic) e dashboard fullstack Next.js. Arquitetura Hexagonal + DDD + Event-Driven.
`Quarkus` `Kafka` `Python` `PostgreSQL` `Next.js` `Anthropic AI`

---

**[Bring Workers](https://github.com/devmarrima)**
Infraestrutura distribuída de coleta e enriquecimento de dados do mercado de trabalho — múltiplos workers especializados, pipeline assíncrono de 8 estágios, publisher Kafka, monitoramento Prometheus e circuit-breakers.
`Python` `FastAPI` `Playwright` `Kafka` `Prometheus`

---

**[DM Catálogo](https://github.com/devmarrima)**
API RESTful de catálogo de produtos com arquitetura em camadas, validação robusta, tratamento global de exceções e padrões gerais de backend enterprise. Demonstra boas práticas de design de APIs, persistência e organização de código.
`Java` `Spring Boot` `PostgreSQL` `JPA` `REST API`

---

**[Hexagonal Orders](https://github.com/devmarrima/Hexagonal-orders-products)**
Sistema de pedidos enterprise com Arquitetura Hexagonal e Domain-Driven Design. Domínio puro sem dependências de framework, ports & adapters para isolamento total de infraestrutura.
`Java` `Spring Boot` `Hexagonal Arch` `DDD` `JUnit 5`

---

**[Gestor Stock](https://github.com/devmarrima/gestor-stock)**
API RESTful de gestão de estoque com controle de movimentações, audit log com timestamps, validação Bean Validation e padrão DTO para contratos de API limpos.
`Java` `Spring Boot` `JPA` `PostgreSQL`

---

**[Employee Auth System](https://github.com/devmarrima/Employee-auth)**
Serviço de autenticação e autorização enterprise com OAuth2, JWT, RBAC e fluxos de refresh token usando Spring Security.
`Java` `Spring Boot` `Spring Security` `OAuth2` `JWT`
