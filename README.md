## Sobre mim

Backend Engineer com foco em **sistemas distribuídos**, **microsserviços** e **arquitetura escalável**. Projeto e construo APIs de alta performance, pipelines orientados a eventos com Kafka, e aplico padrões como Hexagonal Architecture, Clean Architecture e DDD em projetos reais — não em tutoriais.

Atualmente desenvolvendo a plataforma **BringIA**: um sistema distribuído com 6+ microsserviços Quarkus, pipeline de enriquecimento com IA (Anthropic), scraping resiliente e dashboard fullstack em Next.js.

📍 Goiânia, GO — Brasil &nbsp;|&nbsp; 🌍 Disponível para oportunidades remotas
